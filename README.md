# Interior Design Portfolio

A modern, responsive portfolio website for Sarah Johnson Interior Design, showcasing house decoration and customization services. This website demonstrates expertise in interior design, project galleries, and consultation booking.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Elegant, professional design with smooth animations and transitions
- **Interactive Navigation**: Sticky navbar with smooth scrolling and active state indicators
- **Project Gallery**: Beautiful grid layout showcasing interior design projects
- **Services Section**: Organized display of design services and expertise
- **Consultation Booking**: Functional contact form with service selection and validation
- **Performance Optimized**: Efficient animations using Intersection Observer API
- **Accessibility**: Semantic HTML5 structure with proper ARIA labels

## Sections

1. **Hero Section**: Eye-catching introduction with call-to-action for consultation booking

## Customization

### Personal Information
Update the following in `index.html`:
- Name and title in hero section
- About section content with design experience
- Contact information
- Social media links (Instagram, Pinterest, Houzz, LinkedIn)

### Projects
Modify project cards in the projects section:
- Update project titles and descriptions
- Change design style tags
- Add gallery links and detail pages
- Update placeholder images with actual project photos

### Services
Add or remove services in the skills section:
- Organize by category (Interior Design, Home Decoration, Tools & Services)
- Update service icons and descriptions
- Highlight areas of expertise

### Styling
Customize colors and fonts in `css/style.css`:
- CSS variables for easy color changes
- Font family preferences
- Animation speeds and effects

## Images
Replace placeholder images with your own:
- `profile.jpg`: Professional headshot (400x400px recommended)
- `about.jpg`: Design studio or workspace image (500x400px recommended)
- `project*.jpg`: Interior design project photos (400x250px recommended)
